Verdict Hasher

GitHub Repository

These is a CLI hash checking program made with Python. It has support for SHA-1, SHA-224, SHA-256, SHA-384, SHA-512, SHA3-224, SHA3-256, SHA3-384, SHA3-512, MD5, and Blake2B/2S

The aim of this program is to make it easier to generate and/or compare hashes from files, as I often found the process to be a bit too time consuming, at least on Windows.
